# Spec: 의료복지 혜택 검색 및 요약 에이전트 (Welfare Agent)

## 목적
대한민국 보건복지부, 식품의약품안전처, 건강보험심사평가원 등의 공공 복지 정보를 기반으로, 만성질환(콩팥병, 당뇨) 환자들이 처한 경제적·의료적 부담을 경감할 수 있는 실질적인 국가 복지 지원금 및 급여 혜택 정보를 Supabase FTS 데이터베이스 조회를 통해 맞춤형으로 안내합니다.

## 입력
| 파라미터 | 타입 | 필수 | 설명 |
| :--- | :--- | :--- | :--- |
| message | string | Yes | 복지 지원 신청 및 조건 관련 질문 텍스트 |

## 출력
| 필드 | 타입 | 설명 |
| :--- | :--- | :--- |
| answer | string | 한국어로 작성된 혜택 요건, 필요한 서류, 신청 프로세스 요약 |
| agentType | 'welfare' | 고정값 |
| sources | Array | 복지 제공 기관, 공식 사이트 링크 리스트 |

## 처리 흐름 (단계별)
1. **변수 및 질환 태그 추출**
   - 사용자 발화로부터 만성 신부전증 기수 및 당뇨 여부, 필요한 혜택 종류(의료급여, 장애인 등록 등)를 추출합니다.
2. **Supabase FTS BM25 검색**
   - Supabase 데이터베이스의 `search_welfare()` 함수(RPC)를 호출하여 한국어 검색어 적합도를 계산하고 상위 5건의 복지 정책을 검색합니다.
3. **질환 필터링**
   - 추출된 질환 태그(CKD, DM, BOTH)가 데이터에 정의된 질환 분류와 매칭하는지 후처리 필터를 수행합니다.
4. **Pinecone 복지 벡터 폴백 검색**
   - Supabase FTS 검색 결과가 없거나 적합도가 현저히 낮을 경우, `kongdang-welfare` 인덱스를 통해 시맨틱 벡터 유사도로 2차 검색을 실시합니다.
5. **의료 사회복지사 컨설팅 답변 생성 (LLM)**
   - GPT-4o-mini를 사용하여 해당 복지 서비스의 지원 대상 소득 분위, 지원금 한도, 구비 서류(진단서 등), 신청 장소(읍면동 주민센터 등)를 일목요연한 표 및 불릿 리스트 형태로 요약 제공합니다.

## 엣지케이스
- **비정상 소득 기준 및 중복 수혜 여부**:
  - 대처: "본 정보는 보건복지부의 일반적인 기준이며, 가구 소득 비율 및 지자체별 조례에 따라 변동될 수 있으므로 행정복지센터 방문 전 사전 확인을 권장합니다" 문구를 포함합니다.
- **아무 결과가 조회되지 않는 경우**:
  - 대처: 일반적인 만성질환 국가 복지 시스템(차상위 계층 의료급여, 산정특례 제도 등)의 공통 신청 규칙을 안내합니다.

## 성능 목표
- **DB 검색 시간**: 50ms 미만
- **답변 생성 레이턴시**: 2초 이내
- **정책 정확성**: 수치 정보(급여 한도 등) 오답률 0% 지향 (프롬프트 엄격 검증)

## 테스트 케이스 (최소 5개)
1. **의료급여 혜택 검색**:
   - 입력: "만성 신부전 환자 의료급여 1종 2종 자격 조건이 뭔가요?"
   - 기대결과: Supabase 복지 데이터베이스에서 자격 요건 테이블이 검색되어 1종과 2종의 소득 기준 차이가 표 형태로 깔끔하게 요약됨.
2. **장애인 등록 혜택**:
   - 입력: "신장이 안 좋아서 신장 장애인 등록 하려는데 혜택이랑 절차 알려줘"
   - 기대결과: 장애인 연금 및 각종 감면 혜택 목록과 정밀의학진단서 제출 절차가 일목요연하게 안내됨.
3. **DB 접속 에러 폴백**:
   - 입력: Supabase 서비스 일시 다운 시뮬레이션
   - 기대결과: Pinecone 시맨틱 백엔드 폴백 검색을 통해 백업 데이터가 성공적으로 출력됨.
4. **당뇨 소모품 지원비 검색**:
   - 입력: "당뇨 환자 인슐린 주사 바늘이랑 소모품 비용 환급 어떻게 받죠?"
   - 기대결과: 소모성 재료비 환급 제도 설명 및 청구서 작성 서류 안내 제공.
5. **애매한 복지 쿼리**:
   - 입력: "혜택 같은 거 뭐 있어?"
   - 기대결과: 질환 필터를 묻고, 신장병과 당뇨 환자들이 가장 흔하게 신청하는 산정특례 및 본인부담상한액 초과금 환급 제도를 대표로 제시함.
