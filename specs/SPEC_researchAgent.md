# Spec: 연구논문 검색 및 요약 에이전트 (Research Agent)

## 목적
만성 콩팥병 및 당뇨 질환을 지닌 환자들을 대상으로 PubMed E-utilities 검색 API와 Pinecone 백엔드 벡터 데이터베이스를 병용하는 Hybrid RAG 구조를 구축하여, 과학적으로 입증된 신장 치료 연구 및 논문 요약 답변을 제공합니다.

## 입력
| 파라미터 | 타입 | 필수 | 설명 |
| :--- | :--- | :--- | :--- |
| message | string | Yes | 사용자 질문 텍스트 |

## 출력
| 필드 | 타입 | 설명 |
| :--- | :--- | :--- |
| answer | string | 한국어로 작성된 3대 핵심 연구 논문 종합 요약 마크다운 텍스트 |
| agentType | 'research' | 고정값 |
| sources | Array | 각 검색 논문의 제목, 저널명, DOI, URL 리스트 |

## 처리 흐름 (단계별)
1. **변수 추출 (Entity Extraction)**
   - GPT-4o-mini를 이용하여 질환 종류(CKD, DM, BOTH), 병기 단계, 핵심 키워드를 추출합니다.
2. **이중 병행 검색 (Parallel RAG Search)**
   - **PubMed API**: `https://eutils.ncbi.nlm.nih.gov/entrez/eutils/`에 접속하여 최근 5년 이내 논문 중 `(chronic kidney disease OR diabetes) AND {keywords}` 조건으로 최대 10건의 abstract를 로드합니다.
   - **Pinecone 벡터 검색**: `kongdang-papers` 인덱스에 저장된 임베딩을 시맨틱 검색하여 상위 10건의 조각 데이터를 획득합니다.
3. **RRF 병합 (Reciprocal Rank Fusion)**
   - 두 소스에서 가져온 도큐먼트를 제목 및 PMID 유사도 기준으로 병합하고, 순위를 재조정하여 신뢰성이 높은 Top-3 자료를 추려냅니다.
4. **한국어 의료 요약 요약 (LLM Generation)**
   - GPT-4o-mini를 활용하여 환자가 쉽게 이해할 수 있는 어조의 정밀 마크다운 포맷으로 번역 요약문을 생성합니다.
5. **출처 연계 및 반환**
   - PMID 링크 및 DOI 등을 정리한 `AgentResponse`를 반환합니다.

## 엣지케이스
- **PubMed 또는 Pinecone API 중 장애**:
  - 한 쪽이 타임아웃 되거나 장애 발생 시, 정상 작동하는 소스 단독으로 RAG를 수행하여 서비스 연속성을 보장합니다.
- **아무런 검색 매칭 결과 없음**:
  - GPT-4o-mini의 사전 훈련된 의학 지식을 바탕으로 안전 가이드를 설명하되, "정확한 매칭 논문은 조회되지 않았습니다"라는 안내 문구를 상단에 배치합니다.

## 성능 목표
- **검색 매칭 커버리지**: 90% 이상
- **답변 생성 레이턴시**: 2.5초 이내
- **한국어 가독성 지표**: 일반 성인 기준 의학 용어 이해 80% 달성 목표

## 테스트 케이스 (최소 5개)
1. **정상 검색 흐름**:
   - 입력: "만성 신부전증 환자 SGLT-2 억제제 신약 최신 연구 알려주세요"
   - 기대결과: PubMed와 Pinecone에서 관련 논문이 융합되어 요약되고, 출처에 NEJM 혹은 관련 신장 저널 링크가 노출됨.
2. **동작 실패 시 폴백**:
   - 입력: PubMed API 인위적 차단 환경에서 쿼리 실행
   - 기대결과: Pinecone `kongdang-papers`에 저장된 로컬 벡터 문서들만으로 요약이 정상 수행됨.
3. **영문 학술명 쿼리**:
   - 입력: "eGFR stage 4 CKD protect study paper"
   - 기대결과: 영어 키워드가 추출되어 PubMed 검색 성공 및 한국어 번역 요약 완료.
4. **노이즈 및 극단적 질문 필터링**:
   - 입력: "콩팥병을 한 번에 다 고친 기적의 신약 논문 가져와"
   - 기대결과: Safety Checker가 작동하여 차단 답변을 스트리밍함.
5. **완전 무관한 질문**:
   - 입력: "신장 논문 말고 아이돌 컴백 소식 보여줘"
   - 기대결과: `general` 인텐트로 우회되며, 연구 에이전트 미할당으로 비용 절약 처리됨.
